## 1. Introduction
　WAGFはノベルゲームのフレームワークです。文字だけで表現するゲームを作成するためのフレームワークです。絵も音もないゲームの世界です。ただし、絵や音を入れることは可能です。シェアワールドの機能を持っていて、既存のモジュールを1バイトも書き換えることなく、拡張することが出来ます。

　WANGF(Web Autumn's Novel Game Framework)は、かつて存在したANGF(Autumn's Novel Game Framework)をBlazorの技術で再構築しWeb対応させたものです。

　Blazor MAUIの技術を使用し、Web対応版をそのままデスクトップで実行することもできます。その際は、Webの制約を受けないため、オリジナルのANGFに近い機能を使用できます。特にモジュールを読み込む仕組みはWeb版とデスクトップ版では大きく異なるので注意してください。

## 2. How to use
　WANGFの条件を満たすDLLを作成し、それを指定の方法で実行ファイルから参照可能にするとプレイできます。方法はWeb版とデスクトップ版で異なります。

　WNGF SDKはそのためのサンプルソース、技術情報、実行ランタイム、支援ツールを提供します。
